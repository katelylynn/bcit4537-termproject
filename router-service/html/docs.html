Land Bound Drone API

API for sending control commands to a remote-controlled car and tracking user API consumption. This API allows developers to issue movement commands to the car and monitor usage statistics to ensure efficient operation.

get /api-consumption-user
Get API consumption stats for a single user (apiConsumptionUserGet)
Retrieves the total number of requests made by the authenticated user from the database. Includes a warning message if the user has exceeded the free API call limit (20 requests).

Query parameters

Return type

inline_response_200_2

Example data

Content-Type: application/json
{
  "data" : [ {
    "total_requests" : "73",
    "warning" : "Warning! You have passed the 20 free API calls limit."
  }, {
    "total_requests" : "73",
    "warning" : "Warning! You have passed the 20 free API calls limit."
  } ]
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Total requests made by the authenticated user inline_response_200_2

post /transcribe-and-control
Send a voice command to the car (transcribeAndControlPost)
Transcribes an audio file uploaded by the user into a command for the RC car. Returns the transcription, the result of the car command, and any additional response data.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • multipart/form-data

Query parameters

Form parameters

file (required)
Form Parameter — format: binary

Return type

inline_response_200_3

Example data

Content-Type: application/json
{
  "transcription" : "move forward",
  "carResponse" : {
    "status" : "completed",
    "speed" : "10",
    "direction" : "forward"
  },
  "carCommand" : "success"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Transcription and car command executed successfully inline_response_200_3

400

Invalid request or transcription failed validation inline_response_400

500

Internal server error during request processing inline_response_500

Delete user (userDelete)
Deletes the account of the authenticated user from the database. The user's ID (req.user.user.id) is derived from the access_token cookie used for authentication.

Responses
